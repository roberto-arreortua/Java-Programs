## Programas en JAVA

Esta es una recopilación de progrmas en JAVA, forman parte del maual de programación en Java

Se pueden ejecutar directamente en NetBeans o desde la terminal utilizando los siguientes comandos:

1- Nos posicionamos en el directorio donde guardamos el archivo

2- ejecutamos 
```
javac nombre_del_archivo.java
```
3- si no se genera un error
   ejecutamos 

```
java nombre_del_archivo
```
